SVG, The first

Sometimes, CSS is just not worth it. Recently the need arised for a list of social links next to a circular profile image. It would have been nice to curve the link list around the image but such a task in CSS, while very much possible starts to get a bit involved.

You've always got to take any excuse to learn a new skill so I thought this would be a good time to rush up on some working svg knowledge. When it comes to odd shapes and positioning svg simply just has a better tool kit than CSS, mostly due to simpler units and the viewBox.
